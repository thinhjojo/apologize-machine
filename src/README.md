# 🎪 Apologize Machine

> *"Because sometimes sorry just isn't sorry enough."*

**The world's most dramatic apology generator** with support for **4 languages**. Perfect for when you forget birthdays, anniversaries, dates, meetings, or basically anything important.

## ✨ Features

- 🎭 **Dramatic Apologies** — 72+ unique combinations of absurdity
- 🌍 **4 Languages** — English, Tiếng Việt, 中文, 日本語
- 😱 **Extreme Consequences** — from kissing shoes to drinking 1L hot sauce
- 🎲 **Randomly Generated** — no two apologies are the same
- 🍕 **Pizza is Always Eaten Alone** — because priorities

## 🚀 Quick Start

```bash
git clone https://github.com/thinhjojo/apologize-machine.git
cd apologize-machine
python3 apologize_machine.py
```
