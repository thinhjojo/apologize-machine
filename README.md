# Apologize Machine

> *"Because sometimes sorry just isn't sorry enough."*

**The world's most dramatic apology generator** with support for **4 languages**. Perfect for when you forget birthdays, anniversaries, dates, meetings, or basically anything important.

## 📁 Project Structure

```
apologize-machine/
├── src/
│   ├── __init__.py
│   └── main.py          # Main program
├── tests/
│   ├── __init__.py
│   └── test_apologize.py
├── .gitignore
├── CHANGELOG.md
├── CONTRIBUTING.md
├── LICENSE
├── README.md
└── setup.py
```

## 🚀 Quick Start

```bash
pip install .
apologize
```

Or run directly:
```bash
python3 src/main.py
```
