## [1.0.0] - 2024-XX-XX

### 🎉 Initial Release

First public release of the world's most dramatic apology generator.

#### Added
- Multi-language support: English, Vietnamese, Chinese, Japanese
- 16 dramatic apology opening phrases per language
- 25+ absurd reasons for forgetting
- 20 over-the-top actions for atoning
- 16 impossible promises for next time
- 15 questionable consequences if forgotten again
- Interactive CLI with language selection
- ASCII art box output for maximum drama
- MIT License (because lawyers exist)
- README with badges (because open source)
- CONTRIBUTING guide (because community)
- setup.py for pip installation
- .gitignore (because git)

#### Known Issues
- Program does not actually help you remember things
- Capybaras are not included despite being mentioned
- No actual warranty (see LICENSE)
- May cause excessive apologizing in real life
